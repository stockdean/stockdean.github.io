Percona源码安装

@(Database)

  1. github clone Percona-server-5.7
  2. 下载的source code缺少boost，copy mysql的boost到source code
  3. 安装readline和libcurl
    • sudo apt-get install libreadline-dev
    • sudo apt-get install libcurl4-openssl-dev

  4. cmake执行

  5. make
